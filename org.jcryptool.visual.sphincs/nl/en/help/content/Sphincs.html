SPHINCS Signature

This plugin shows how the post-quantum signature algorithm SPHINCS works.

At the moment, this English online help is only a stub. In the German version the usage and the algorithm are already described in detail. Below you will find further information about the algorithm from external sources.

Further information

There is an official website about the SPHINCS algorithm, where you can find papers, presentations and software about SPHINCS.

The SPHINCS structure is based on Merkle trees, which you can also find in JCrypTool (see linked below).
