Demonstration der Punktaddition auf elliptischen Kurven

Die Demonstration veranschaulicht die Anwendung fundamentaler Operationen auf elliptischen Kurven in der Kryptographie. Elliptische Kurven können ausgewählt werden und diese dann manipuliert werden.

Arten von Kurven

Es gibt drei Arten von Kurven:
1. Kurven in der Ebene der reellen Zahlen
2. Kurven im Feld F(p)
3. Kurven im Feld F(2^m)

1. Kurven in der Ebene der reellen Zahlen

Überblick

Elliptische Kurven in der Ebene von reellen Zahlen entsprechen der Gleichung y^2 = x^3 + a*x + b, wobei die Variablen a, b, x and y reelle Nummern sind.
Auf der reellen Zahlenebene gibt es unendlich viele Punkte auf der Kurve. Für jeden Punkt auf der Kurve gibt es einen negativen Wert, so das y = -y mit der Ausnahme y = 0.

Zwei Punkte auf der Kurve können einfach addiert werden:
1)Wähle einen Punkt P und einen Punkt Q.
2)Male eine Linie durch P und Q. Fü diese Linie fort, bis die Kurve ein drittes Mal durchschnitten wird (P und Q sind die ersten zwei Male). Dieser Punkt ist -R.
3)Punkt R ist der Punkt -R mit invertierten y-Wert.

y² = x³ - 10x + 15

Berechnungen

Punktaddition
Punkt R ist Punkt O, das Fernelement, wenn die Gleichung P = -Q erfüllt wird. In diesem Fall, durchschneidet die Linie die durch P und Q führt die Kurve nie ein drittes Mal.

Wenn P != Q:
s = (Yp - Yq) / (Xp - Xq)
Xr = s^2 - Xp - Xq
Yr = -Yp + s(Xp - Xr)
Punktmultiplikation
Wenn Yp = 0, dann ist R = O.

2P = P + P = R
s = (3Xp^2 + a) / (2Yp)
Xr = s^2 - 2Xp
Yr = -Yp + s(Xp - Xr)

2. Kurven im Feld F(p)

Überblick

Eine Kurve im Feld F(p) kann durch die Gleichung (y^2) mod(p) = (x^3 + a*x + b) mod (p) beschrieben werden.
Die Variablen a, b, x und y sind Elemente von F(p) und p ist eine Primzahl. Die Anzahl von Punkten im Feld F(p) ist begrenzt auf p*p Punkte (endlicher Körper).

y² mod 29 = (x³ + 14x + 15) mod 29

Berechnungen

Punktaddition

Punkt R ist Punkt O wenn P = -Q erfüllt wird.

Wenn P != Q:
s = ((Yp - Yq) / (Xp - Xq)) mod p
Xr = (s^2 - Xp - Xq) mod p
Yr = (-Yp + s(Xp - Xr)) mod p

Punktmultiplikation
Wenn Yp = 0, dann ist R = O.

2P = P + P = R
s = ((3Xp^2 + a) / (2Yp)) mod p
Xr = (s^2 - 2Xp) mod p
Yr = (-Yp + s(Xp - Xr)) mod p

3. Kurven im Feld F(2^m)

Überblick

Elemente im Feld F(2^m) sind m-Bit Strings. Kalkulationen befolgen die Regeln der Polynomarithmetik.
Die Berechnungen können von Computern effizient berechnet werden, da F(2^m) Operationen auf Bitfolgen ausfürt.
Die binäre Repräsentation einer elliptischen Kurve kann durch die Gleichung y^2 + xy = x^3 + ax^2 + b beschrieben werden, wobei a, b, x and y Elemente des Feldes F(2^m) sind.
Das Feld F(2^m) wird mithilfe des Generators G definiert. G ist ein irreduzibles primäres Polynom mit der Bitlänge m + 1.
Das Negativ von Punkt P = (Xp, Yp) ist -P = (Xp, Xp + Yp).

y² + xy = x³ + g9*x² + g3 ; generator G = 110111 ; m = 5

Berechnungen

Polynomberechnungen
Es ist wichtig Kalkulationen mit Polynomen zu verstehen, um Berechnungen über F(2^m) zu verstehen.
In den folgenden Beispielen werden alle Polynom als Bitfolgen dargestellt (e.g. for m=5 : x^4 + x^2 + x + 1 = 10111)

Addition (mithilfe XORs)
1011 + 0101 = 1110

Multiplikation
1011 * 1 = 1011
1011 * 10 = 10110
1011 * 11 = 10110 + 1011 = 11101

Division

Punktaddition

Punkt R ist Punkt O wenn P = -Q.

Wenn P != Q:
s = (Yp - Yq) / (Xp + Xq)
Xr = s^2 + s + Xp + Xq + a
Yr = s(Xp + Xr) + Xr + Yp

Punktmultiplikation

Wenn Xp = 0 hält, dann ist R = O.

2P = P + P = R
s = Xp + Yp / Xp
Xr = s^2 + s + a
Yr = Xp^2 + (s + 1) *Xr
